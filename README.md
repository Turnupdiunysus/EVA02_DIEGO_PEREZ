# EVA02_DIEGO_PEREZ

Este programa en Python calcula rutas entre dos lugares usando el servicio de mapas Graphhopper.

¿Qué hace?
Pide al usuario un punto de inicio y un destino.

Consulta la API de Graphhopper para encontrar la mejor ruta.

Muestra la distancia, el tiempo estimado y las instrucciones paso a paso.

¿Cómo usarlo?
Tener Python instalado y conexión a internet.

Ejecuta en la terminal:

bash
python3 eva02_perez_graphhopper.py
Sigue las instrucciones en pantalla.

Escribe (s) o (salir) para terminar el programa.
